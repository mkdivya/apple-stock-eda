# 📊 Apple Stock Market EDA

## 📌 Objective

To analyze Apple stock data using Exploratory Data Analysis (EDA) techniques and extract meaningful insights.

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* yfinance

## 📂 Features Implemented

* Data collection using API (yfinance)
* Data cleaning and preprocessing
* Feature engineering (returns, log returns, moving average, volatility)
* Data visualization
* Correlation analysis

## 📊 Key Insights

* Stock shows overall upward trend after 2020
* Moving average helps identify trends clearly
* Volatility highlights risk periods
* Returns distribution is centered near zero

## 🚀 How to Run

1. Install dependencies
2. Run notebook step by step
3. Analyze outputs

## 📁 Output

* Clean dataset
* Visualizations
* Insights
